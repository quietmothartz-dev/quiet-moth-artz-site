# Quiet Moth Artz Framer Free Build Guide

Use this guide to rebuild the Quiet Moth Artz website in Framer Free today. This is a no-code build plan based on the existing `quiet-moth-artz-site` content blueprint.

## Site Goal

Create a clean, professional one-page website for Quiet Moth Artz that promotes creative consulting, website design, graphic design, songwriting/hooks, brand direction, automated creative workflows, and digital creative systems.

## Page Structure

Create one Framer page with these sections in order:

1. Header / Navigation
2. Hero
3. Services
4. Portfolio / Featured Work
5. Free AI Hook Generator
6. Website Packages
7. Why Work With Me
8. Hire Me / Contact
9. Footer

## Global Style

### Visual Direction

Use a premium creative consultant look:

- Dark creative background
- Clean white or off-white content sections
- Soft glow/spotlight accents
- Moth-inspired atmosphere without making it cartoonish
- Professional spacing and large readable type
- Polished service, portfolio, and package cards

### Suggested Colors

- Main dark background: `#100D18`
- Secondary dark background: `#171122`
- Light section background: `#FFFDF8`
- Cream text/accent: `#FFF8E9`
- Main text on light sections: `#121018`
- Muted text: `#756F7D`
- Gold accent: `#D9A441`
- Purple accent: `#8E67FF`
- Soft rose accent: `#D96583`
- Dark card border: rgba black at 10-15%
- Light card border: rgba white at 12-18%

### Font Style

Use a clean modern sans-serif font. Good Framer font choices:

- Inter
- Satoshi
- Manrope
- Space Grotesk

Recommended:

- Headings: Space Grotesk or Satoshi, bold
- Body: Inter or Manrope, regular/medium
- Buttons: same font, bold

### Spacing

Desktop:

- Page max width: about 1120-1240px
- Section padding: 90-120px top and bottom
- Side padding: 48-72px
- Card gap: 18-24px
- Hero should take most of the first viewport

Mobile:

- Side padding: 20px
- Section padding: 64-80px top and bottom
- Stack all cards in one column
- Buttons should be full-width or comfortably tappable
- Keep headline large but not cramped

## Free AI Hook Generator Section

### Anchor

Set this section ID/name to `hook-generator`.

### Important Framer Free Note

For Framer Free today, this section should be a promotional section, not a working embedded app unless you use an external embed later. Build it as a featured tool preview with a call to action.

### Layout

Desktop:

- Left column: title, text, CTA sentence
- Right column: polished mock generator card

Mobile:

- Stack text above mock generator card

### Copy

Title:

Free AI Hook Generator

Text:

Need a hook for a song, campaign, slogan, or creative idea? Try my free hook generator, then book Quiet Moth Artz if you want help turning the idea into a finished song, brand, flyer, or website.

CTA sentence:

Want me to write the full hook, song concept, or brand slogan for you? Book a creative session.

### Mock Generator Card

Create a card that visually shows:

- Input label: Topic, artist name, brand name, song idea, or phrase
- Example input: summer single
- Dropdown label: Style / mood
- Example selected option: Street anthem
- Button: Generate Hook
- Button: Copy Hook
- Button: Generate Again
- Dark output area with 2-3 sample hook ideas

Sample output text:

Hook idea 1: Built from the pressure, summer single shining through the climb.

Hook idea 2: Lights up, hands high, summer single in the room tonight.

Hook idea 3: Clean design, clear message, summer single made to promote.

### Future Button

Add a button:

Try the Hook Generator

For now, link it to the contact section or leave it ready to link later. When the Cloudflare/Codex Hook Generator is live, replace this button link with that URL.

## Fast Launch Checklist for Framer Free

1. Create a new Framer site.
2. Build one page with the sections listed in this guide.
3. Add the sticky header/navigation.
4. Add the hero copy and buttons.
5. Create the 6 service cards.
6. Create the 6 portfolio cards with placeholder visual blocks.
7. Create the Hook Generator promo/mockup section.
8. Create the 3 website package cards.
9. Add the Why Work With Me checklist.
10. Add the Hire Me / Contact section with the mailto button.
11. Add the footer.
12. Check desktop layout.
13. Check tablet layout if available.
14. Check mobile layout.
15. Test every navigation link.
16. Test the email button.
17. Open Preview and read every section for typos.
18. Publish on Framer Free.
19. Copy the Framer published URL.
20. Share the link and start promoting.

## Linking to the Cloudflare/Codex Hook Generator Later

When the working Hook Generator is ready as a separate Cloudflare Pages project:

1. Deploy the Hook Generator project to Cloudflare Pages.
2. Copy the live Cloudflare Pages URL.
3. In Framer, go to the Free AI Hook Generator section.
4. Select the `Try the Hook Generator` button.
5. Change the link from the contact section to the live Cloudflare URL.
6. Set the link to open in the same tab if it feels like part of the site.
7. Set the link to open in a new tab if it is a separate tool page.
8. Update the section text if needed:

Try the free browser-based Hook Generator, then book Quiet Moth Artz if you want help turning the idea into a finished song, slogan, flyer, brand, or website.

Optional future structure:

- Framer site: main brand website
- Cloudflare Pages: working Hook Generator app
- Framer button: links directly to the Cloudflare Hook Generator

## Final Launch Note

Framer Free is a good choice for getting the brand online quickly. Keep Version 1 simple: clear offer, polished visuals, strong service cards, direct email button, and a visible path to book creative work.
